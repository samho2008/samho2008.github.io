# ASCII

ASCII（American Standard Code for Information Interchange）美国信息交换标准代码，从 1967 年发表，现在计算机的基础编码。总共有 128 个，对所有的英文字符及部分特殊符号进行了编号。

# Latin-1

总共有 256 个，对欧洲字符进行了编码，兼容 ASCII。MySQL 的默认使用编码。

# GB2312

用于存储中文字符。对常用的六千多汉字进行了编码，兼容 ASCII。

# GBK

对两万多汉字进行编码，兼容 GB2312（GB2312 兼容 ASCII）。

# BIG5

台湾繁体字编码

# Unicode

对世界上主流国家常用的语言进行了编码，兼容 ASCII，具体包含了 3 种存储方案，分别是 utf-8、utf-16、utf-32。

# UrlEncode/Decode

针对网络中传输，在路径中传输数据，需要进行 UrlEncode/UrlDecode 编码解码。
