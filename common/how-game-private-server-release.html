# 网络游戏私服行业现状和反私服的措施

# 私服行业现状

# 私服是怎么来的？

简单点说就是服务端泄漏了。

至于泄漏的原因，那就太多了，有服务端程序泄漏，也有源码泄漏的。

# 服务端程序文件泄漏

  1. 国外游戏，中国代理，可执行文件泄漏。像奇迹MU和传奇我估计都是这种情况，早期大家防范意识不够，合理的方式应该是不让代理商接触服务端。

  2. 服务器被黑客攻击，拿走了可执行文件。曾经奇迹世界SUN有一个版本据说就是这样泄漏的。

  3. 内部人员泄漏。

# 服务端源码泄漏

  1. 这种大部分是内部程序员离职时带出去的，程序员离职时，一般会把代码带走，这件事怎么说呢，可以理解，毕竟写了那么久了，以后也有用得着的代码，不带走多可惜，跟白干几年差不多。带走就带走吧，不传播也没事，但有些人就是不自觉，把代码发给别人，这下基本上就一传十、十传百了（传播速度很快）。

  2. 还是黑客攻击。G胖的邮箱曾被一个黑客攻击，把反恐精英的代码偷走了。

# 执法不严

由于私服太多了，不太可能一一打击，这就给了很多私服业主可乘之机。

而且官服在打击私服时，要花很多钱（要出差取证，出差费、取证费等等），而最终抓到私服后罚的钱，全部上交国家了，官服不会收到一分钱。所以对于很多不那么赚钱的游戏来说，对私服都是睁一只眼闭一只眼的。

警察这边也是一样的，他们也是多一事不如少一事，像这种事，他们一般也不爱管，也不会主动去管。都是官方运营报案了，他们去会去管一管。而且如果游戏公司和当地警方关系不好的话，就算是报案可能也不爱管。

举个例子： 以前参与开发的某端游，一个月收入只有80万RMB左右，抓一次私服可能要花10万，运营的绩效不会因为抓了私服就会有加分，只会看钱，少了10万，那就给你个B。

# 私服业主

  • 纯粹以赢利为目的的人群（大部分都是这种）

  • 对某游戏有深厚感情，想开个玩玩，不以赢利以目的（有情怀的玩家，虽犯罪，但由于没有赢利，处罚较轻，甚至不处罚，只勒令关闭）暴雪勒令世界最大的魔兽世界私服Nostalrius关闭

  • 群服（QQ群里一帮爱玩的老玩家搭一个玩，不对外运营，不犯罪，也没人管）

  • 自嗨的就不算了，这种是当单机游戏玩的（可以在淘宝上搜到各种游戏的单机版）

  • 天堂

特点：

  • 早期端游（那时候保护意识还不强）
  • 都是当年非常成功的游戏（不成功的玩家基数太小）

可以看到韩国游戏偏多，这个也是有原因的。 私服业主在开私服时，是不管你是韩国还是中国的，只要玩家多，能赚钱，就会去开。但韩国的相对安全一点，尤其是中国没有代理或代理已死的，这样的游戏是绝对没有人管的（不要幻想ZF会去管你，哪那么多闲心管你）

# 收银系统

  • 早期，纯手工转账，截图，再手动发装备

  • 自动发卡平台（卖卡密）

  • 淘宝店

  • 前几年还可以用第三方支付平台（易宝、灵风易等等），现在国家管的严了，要提供营业执照，这些都不好用了。

  • 开一个壳公司，弄个营业执照，表面上做其他业务，背地里运营私服。

# 游戏内怎么赚钱

  • 高级点的，跟传统官服一样，道具商城卖钱，游戏内的经济系统比较健康，平衡性也可以，游戏寿命长。

  • 低级点的，直接明码标价卖强力装备，游戏没有任何平衡性可言。

传奇私服市场规模保守估计一年在30亿-50亿之间 (opens new window)

# 租服务器

国内的，小厂商，做灰色业务。大厂一般不做私服，像阿里云腾讯云这种，一旦发现马上就封。

还有一点，私服容易被DDoS攻击，会让整个机房的服务器都受影响，也是严重威胁着他们的服务质量，从这一点上来说，他们也不愿意为私服业主提供服务。

国外的网络不太行，除非做国外市场。

有云服和物理机两种，物理机性能会好点，云服省心点。

# 私服怎么推广

  • 私发网 (opens new window), 全名叫私服发布网（要花钱，效果也好）
  • 论坛、QQ群发小广告
  • 利用玩家贪小便宜的心理，利用玩家推广，手段一般是把他们提供的小广告转发N个群，截图后可以获得什么装备
  • 有玩家资料的专业运营，也会定点推广（手机短信、邮件），这种效果是最好的，但是获取用户资料这件事难度比较高（一种是开N年的私服业主，一种是官方运营）

怎么推广这件事，还是要各显神通。

# 反私服措施

# 物理手段

  • 锁机箱、建专网（某游戏公司就这么干的，成本比较高，也不方便）

  • 代码权限控制（最少接触原则）

  • 爱国主义教育

  • 报警、抓抓抓

# 软件手段

  • 服务端加壳、资源加密

  • 启动时增加验证机制，证书授权等等

  • 认证服务器，定时认证

  • 建立反私服平台，服务端留后门，一旦发现，上报IP到平台

# 手游私服的情况

现在的游戏很少有私服了，原因就是手游当道，游戏太多，玩家“阅游无数”，很难对某一款游戏产生感情，玩几个月就丢了。

而且现在游戏开发不是难事，最难的是运营了，官方都还发愁去哪买量呢，私服业主去哪找人玩？

当然也不是没有，以前见过Dota传奇的私服。
