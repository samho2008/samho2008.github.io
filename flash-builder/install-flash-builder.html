# 安装 Flash Builder 4.7

Flash Builder 4.7 是 Adobe 发布的最后一个版本。

官方下载地址：https://prdl-download.adobe.com/Flash%20Builder/C06A256DD8FD4DF186D90CE0ADCF0B2C/1509477139530/FlashBuilder_4_7_LS10_win64.exe (opens new window)

镜像下载地址：

下载【FlashBuilder_4_7_LS10_win64.exe】，双击运行。

等待解压完成。

在桌面找到解压后的目录。

运行里面的【Set-up.exe】。

先点击【试用】完成安装，后面再进行破解。

输入邮箱和密码后点击登录，没有的话可以去注册一个。

勾选上所有组件，然后点击【安装】。

下载完成后，点击【关闭】。

# 复制 AIR SDK

默认的 AIR SDK 位置在【C:\Program Files\Adobe\Adobe Flash Builder 4.7 (64 Bit)\eclipse\plugins\com.adobe.flexbuilder.flex_4.7.0.349722\devsdks\AIRSDK\Win】，而且还不能改，但是 FlashBuilder 自带的 AIR SDK 又不是在这里，因此要把其他地方的 AIR SDK 复制过来。

在【C:\Program Files\Adobe\Adobe Flash Builder 4.7 (64 Bit)\eclipse\plugins\com.adobe.flexbuilder.flex_4.7.0.349722】下新建目录【devsdks/AIRSDK/win】，然后把【C:\Program Files\Adobe\Adobe Flash Builder 4.7 (64 Bit)\eclipse\plugins\com.adobe.flash.compiler_4.7.0.349722\AIRSDK】里面的内容全都复制过来。

# 破解 FlashBuilder 4.7

注意：破解前先关闭 FlashBuilder。

进入目录【C:\Program Files\Adobe\Adobe Flash Builder 4.7 (64 Bit)\eclipse\plugins\com.adobe.flexbuilder.project_4.7.0.349722\META-INF】，修改【MANIFEST.MF】文件。

搜索【Bundle-Version】，改成【Bundle-Version: 0.0.0】，改完记得保存。

进入目录【C:\Program Files\Adobe\Adobe Flash Builder 4.7 (64 Bit)\eclipse\features\com.adobe.flexide.feature_4.7.0.349722】，修改【feature.xml】。

搜索【id="com.adobe.flexbuilder.project"】，把 version 改成【0.0.0】。

进入目录【C:\Program Files\Adobe\Adobe Flash Builder 4.7 (64 Bit)\eclipse\plugins\com.adobe.flexbuilder.flex_4.7.0.349722】，把里面的【config.xml】复制一份，然后重命名为【config_builder.xml】。

再次启动 FlashBuilder，就没有弹窗提示了。
