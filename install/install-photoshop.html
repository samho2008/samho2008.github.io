# 安装 Photoshop

# 下载 Photoshop

下载地址：「Adobe全家桶2022」https://www.aliyundrive.com/s/Ev7N1rAcxLN 提取码: 6j1j 点击链接保存，或者复制本段内容，打开「阿里云盘」APP 。

下载里面的【PS 2022.exe】

# 解压 PS 2022.exe

双击【PS 2022.exe】，选择解压路径。

这里只是把要安装的安装包解压出来，并不是实际安装，而且会自动创建下载目录。

# 运行 Set-up.exe

解压完成后，进入解压后的目录【Adobe Photoshop 2022 SP】，双击里面的【Set-up.exe】。

点击继续，就开始安装了，耐心等待安装完成。

安装完成后，点击【关闭】。

# 运行 Photoshop

安装完成后，桌面会多了一个【Adobe Photoshop 2022】的图标，双击即可运行，或者点击左下角开始菜单找到【Adobe Photoshop 2022】运行。
