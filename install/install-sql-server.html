# 安装 SQL Server

# 安装 SQL Server 2019

# 安装环境

  • 操作系统：Windows Server 2016 Datacenter（版本 1607，OS 内部版本 14393.447）

# 1. 下载数据库镜像文件

cn_sql_server_2019_enterprise_x64_dvd_2bfe815a.iso

下载地址：ed2k://|file|cn_sql_server_2019_enterprise_x64_dvd_2bfe815a.iso|1632086016|58C258FF0F1D006DD3C1F5F17AF3EAFD|/

也可以通过官网下载：

https://www.microsoft.com/zh-cn/sql-server/sql-server-downloads (opens new window)

# 2. 安装镜像文件

# 2.1 双击进入镜像文件目录

进入目录后找到【setup.exe】，右键【以管理员身份运行】，避免因权限问题导致的安装失败。

在左侧找到【安装】，然后选择第一个【全新 SQL Server 独立安装或向现有安装添加功能】。

Microsoft Sql Server 2019 Enterprise 产品密钥：【HMWJ3-KY3J2-NMVD7-KG4JR-X2G8G】

勾选【数据库引擎服务】，然后点击下一步。

一般来说勾选这个就足够了，其他的使用到了再来安装也行。（大多数情况下都用不到）

建议勾选以下功能，后面再装挺麻烦的。

勾选【数据库引擎服务】、【SQL Server 复制】、【全文和语义提取搜索】、【Data Quality Services】、【客户端工具连接】、【客户端工具向后兼容性】、【客户端工具 SDK】、【SQL 客户端连接 SDK】。

一般情况下，其实只选这个也可以。

以后要用到其他在安装。

选择【混合模式】，然后【输入密码】，点击【添加当前用户】作为 SQL Server 管理员，然后点击【下一步】。

安装完成后点击【关闭】。

把【SQL Server 安装中心】也关闭掉。

# 2.2 安装 SSMS (SQL Server Management Studio)

安装完数据库后，如果想连接数据库，还需要另外安装一个可视化的界面。

打开官网：https://docs.microsoft.com/en-us/sql/ssms/download-sql-server-management-studio-ssms?view=sql-server-ver15 (opens new window)

向下滚动页面找到【Download SSMS】，然后点击【Free Download for SQL Server Management Studio (SSMS)】。

下载完成后，右键【以管理员身份运行】。

点击【安装】。

等待安装完成，安装完成后点击【重新启动】，或者稍后自行重启。

# 3. 连接 SQL Server

在左侧开始菜单找到【Microsoft SQL Server Management Studio】并点击运行。

选择【SQL Server 身份验证】确保【sa】用户可以正常登录。
