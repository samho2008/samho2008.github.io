# 安装 WinRAR

WinRAR 下载地址：https://gitlab.com/toolcenter/tool/-/raw/main/winrar/6.11/winrar-x64-611sc.exe?inline=false (opens new window)

ResourceHacker 下载地址：https://gitlab.com/toolcenter/tool/-/raw/main/resource-hacker/5.1.7/resource_hacker.zip?inline=false (opens new window)

虽然 WinRAR 可以免费使用，但是每次打开都会弹出广告，怎么把这个广告去掉呢？

备注：只有简体中文版会有广告，和英文版会提示你购买，而且简体中文即使有了许可文件也会弹出广告，英文版只需要有许可文件就可以直接完成注册了！

需要用到工具 ResourceHacker，在【C:\Program Files\WinRAR】找到【WinRAR.exe】，复制一份，防止修改坏了还能还原回来继续改。

把【WinRAR.exe】拖进【ResourceHacker】里，然后找到【String Table】

然后找到 1277 这一行，双击选中，然后按【DEL】键把整行删除掉。

改成如图所示的样子。

点击上方绿色的按钮，重新编译下软件。

看到按钮变成灰色以后，点击左上角的【File】，选择【Save AS...】。

注意：保存到桌面，不要直接覆盖原来目录下的，不然会提示没有权限！

然后把桌面的【WinRAR.exe】拖到安装目录下，覆盖掉原来的。

再次打开，广告就没有了，但是会有许可的提示。

这个时候需要把许可文件【rarreg.key】，复制进安装目录，然后重新打开【WinRAR.exe】，没有看到提示说明破解成功了。
