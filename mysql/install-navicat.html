# 安装 Navicat

# Windows 系统安装 Navicat

# Navicat 16

# 下载 Navicat 16

下载 Navicat 16：https://gitlab.com/samho2008/tool/-/raw/main/navicat/navicat16/navicat160_premium_cs_x64.exe (opens new window)

# 安装 Navicat 16

双击运行下载好的 Navicat 16，安装过程非常简单，一直下一步就可以了。

等待安装完成。

# 破解 Navicat 16

由于 Navicat 是付费软件，想要永久使用就需要进行破解。

# 0. 运行破解工具

破解工具下载地址：https://gitlab.com/samho2008/tool/-/raw/main/navicat/navicat16/NavicatCracker%20v16.0.7.0.exe (opens new window)

右键【以管理员身份运行】运行下载好的破解工具。

注意：必须【以管理员身份运行】，因为默认安装到 C 盘，不以管理员身份运行破解工具没有权限操作 Navicat。

# 1. 破解 Navicat 程序

这一步会对安装包中的【libcc.dll】进行破解修改，而因为安全设置对 C 盘里面文件的操作是需要用管理员权限的，所以需要【以管理员身份运行】的方式启动破解工具。

可以看到，修改后安装包下会多了一个【libcc.dll.bak】。

破解完成后，会出来一个对话框，问你是否要导出私钥。这里必须选【是】。

# 2. 生成 KeyGen

点击 Generate，生成 KeyGen，其他参数不要修改。

打开桌面上的 Navicat。

把工具生成出来的 KeyGen 复制到 Navicat 中。

这个时候，我们会看到这个提示。不要慌，我们需要断网激活。

点击右下角的网络。

再次激活。

# 3. 生成激活码

看到这个提示，说明破解成功了。

# 初始化 Navicat 16

第一次打开，会给你介绍这个版本多了什么新功能。

等待 3 秒后，才可以选择。注意，这里选择【不共享】。

看到永久许可证，说明破解成功了。

# 恢复网络

破解 Navicat 的时候，我们使用离线激活，把网卡禁用掉了，记得恢复回来不然没法正常上网了。

# 删除 Navicat_2048bit_rsa_private_key.pem

用工具破解 Navicat 的时候会在工具的当前目录下生成私钥文件【Navicat_2048bit_rsa_private_key.pem】，破解完成之后这个文件就不需要了，可以删掉。

# 禁用 Navicat 16 自动更新

毕竟是破解版，防止因为版本更新导致不可用，因此需要禁用掉自动 Navicat 的自动更新。

在上方菜单栏点击【工具】，选择【选项...】。

把【在启动时自动检查更新】前面的勾去掉。

# Navicat 15

# 下载 Navicat 15

下载 Navicat 15：https://gitlab.com/samho2008/tool/-/raw/main/navicat/navicat150_premium_cs_x64.exe (opens new window)

# 安装 Navicat 15

双击运行下载好的 Navicat，安装过程非常简单，一直下一步就可以了。

等待安装完成

# 破解 Navicat 15

由于 Navicat 是付费软件，想要永久使用的话需要进行破解。

# 运行破解工具

下载破解工具 Navicat Keygen Patch v5.6.0 DFoX.exe，下载地址：https://gitlab.com/samho2008/tool/-/raw/main/navicat/Navicat%20Keygen%20Patch%20v5.6.0%20DFoX.zip (opens new window)

下载好之后，把软件解压出来。

解压后，双击运行解压后的软件。

（如果解压出来之后注册软件被 Windows Defender 删除了，需要先把 Windows Defender 关闭）

如果有安全警告，选择是。

# 破解安装包

注意：破解之前请确保不要运行 Navicat！

首先，点击右上角的 Patch（此步骤会对安装好的软件进行破解，生成 RSA 证书，如果没有操作这个步骤会导致后面破解失败）。

等待一会后，软件破解完成，看到以下提示说明破解成功了。

# 生成 Serial Keygen

点击 Generate 生成 Serial Keygen，然后点击 Copy。

# 运行 Navicat

打开桌面上的 Navicat Premium 15

点击注册

复制从注册机生成出来的密钥，然后点击激活。

如果提示激活失败，激活次数达到上限，需要先断开网络。

Win + i打开设置，选择网络和 Internet

禁用网络

回到 Navicat，再次激活，选择手动激活。

复制 Navicat 中的请求码到注册机中，然后点击 Generate 生成激活码，最后把生成的激活码复制到 Navicat 中，点击激活即可。

看到这个对话框说明破解成功了。

点击确认。

第一次打开会给你介绍有什么功能，选择下一步即可。

根据自己喜好选择白色或黑色主题。

下一步，不使用 Navicat Cloud。

这里需要等待三秒之后，才可以点击，选择不共享。

初始化完成。

# 恢复网络

破解的时候需要关闭网络断网激活，破解完成之后记得开启网络，否则无法正常上网。

在网络连接中右键启用网络。

# 常见问题及解决方案

# 生成 Activation Code 的时候提示 Rsa Public Key not find..

如图所示，点击 Generate Activation Code 的时候出现这个提示。

# 原因分析

这是因为缺少了做第一步 Patch，对软件进行破解。

# 解决方案

点击第一步的 Patch

# Patch 的时候提示 No All Pattern Found! File Already Patched?

# 原因分析

Navicat 正在运行（已经打开），无法进行破解。

# 解决方案

关闭正在运行的 Navicat
