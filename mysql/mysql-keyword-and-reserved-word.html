# MySQL 关键字和保留字汇总

在使用 MySQL 的时候，一般尽量避免用关键字作为表名和字段名。

如使用关键字做表名或字段名，在写 SQL 语句的时候需要添加反引号 ` 来区分关键字和字段名或表名。
