# 临时邮箱

临时邮箱用于注册一些不常用的网站、博客、论坛等，这些网站需要使用电子邮箱的验证码或者发送软件下载链接到电子邮箱中，这个时候可以使用临时邮箱避免被垃圾邮件骚扰，避免个人信息泄漏。

注意：临时邮箱过一段会失效，甚至网页刷新就失效，如果是重要的网站不要使用临时邮箱！！
