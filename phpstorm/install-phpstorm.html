# 安装 PhpStorm

# Windows 安装 PhpStorm

# 方式一：exe 安装包方式安装 PhpStorm

# 1. 下载 PhpStorm

下载 PhpStorm：https://www.jetbrains.com/phpstorm/download/other.html (opens new window)，找到【Version 2020.1】，然后选择【2020.1.3】这个版本。

# 2. 安装 PhpStorm

双击运行下载好的【PhpStorm-2020.1.3.exe】

点击【Next】进入下一步。

使用默认的安装路径即可，不建议修改。

勾选【64-bit launcher】，然后点击【Next】。

点击【Install】下一步，这里不需要修改。

等待安装完成。

点击【Finish】完成安装。

# 3. 配置 PhpStorm

在桌面双击运行安装好的 PhpStorm。

勾选同意协议，不然没法点击 Continue

不发送隐私数据，选择【Don't Send】。

选择【Light】主题，然后选择【Skip Remaining and Set Defaults】

选择【Evaluate for free】，然后点击【Evaluate】。

# 4. 破解 PhpStorm

把【jetbrains-agent.jar】拖进 PhpStorm

选择【Restart】

重启完成后，会出现以下界面，选择【为 PhpStorm 安装】。

选择【是(Y)】

点击右下角的【Configure】

选择【Manage Licenses...】

看到以下界面说明破解成功了。

# 方式二：zip 压缩包方式安装 PhpStorm

# 1. 下载 PhpStorm

下载 PhpStorm：https://www.jetbrains.com/phpstorm/download/other.html (opens new window)，找到【Version 2020.1】，然后选择【2020.1.3】这个版本。

# 2. 解压 PhpStorm 压缩包

解压下载好的压缩包【PhpStorm-2020.1.3.win.zip】

解压后进入【bin】目录，修改【idea.properties】配置文件，

把【idea.config.path】、【idea.system.path】这两个属性复制下来，修改为具体的安装路径，

把【idea.plugins.path】、【idea.log.path】前面的 # 注释去掉，修改完成后记得按 Ctrl + S 保存。

进入【bin】目录，修改【phpstorm64.exe.vmoptions】配置文件，根据实际情况修改以下内容：

  • -Xms：启动时使用的堆内存，建议改为 2048m
  • -Xmx：最大使用的堆内存，建议改为 4096m
  • -XX:ReservedCodeCacheSize：保留的代码缓存大小，建议改为 1024m

修改完成后记得按 Ctrl + S 保存。

# 3. 配置 PhpStorm

双击运行【bin】目录下的【phpstorm64.exe】

第一次打开会出现这个弹窗，勾选同意协议，不然没法点击 Continue

不发送隐私数据，选择【Don't Send】。

选择【Do not import settings】不导入配置。

选择【Light】主题，然后选择【Skip Remaining and Set Defaults】

先选择试用。

# 4. 破解 PhpStorm

把【jetbrains-agent.jar】拖进去。

点击【Restart】重启下使插件生效。

点击【为 PhpStorm 安装】

因为用的是免安装版本，到这里还没有破解完。

打开的窗口也关闭先。

进入文件夹【C:\Users\Public.jetbrains】，可以看到这里有两个文件【important.txt】和【jetbrains-agent-v3.2.1.c46b.ed7】，把这两个文件都复制到【PhpStorm】的【bin】目录下。

接下来要修改【phpstorm64.exe.vmoptions】，通过 JavaAgent 代理的方式来让 PhpStorm 启动的时候去自动加载插件。

在【phpstorm64.exe.vmoptions】配置文件最后添加以下内容：

-javaagent:/PhpStorm-2020.1.3.win/bin/jetbrains-agent-v3.2.1.c46b.ed7=by https://zhile.io

注意：这里不能使用相对路径，比如【./jetbrains-agent-v3.2.1.c46b.ed7=by https://zhile.io】，必须使用绝对路径，不然会启动不起来。

修改完成后保存，然后再次启动【phpstorm64.exe】。

点击右下角的【Configure】，选择【Manage Licenses...】

看到以下界面说明破解成功了。

提示：之前破解用到的目录【C:\Users\Public.jetbrains】可以删除掉。

# 配置 Php 环境

# 1. 下载 Php SDK

打开 https://windows.php.net/downloads/releases/archives/ (opens new window) 下载【php-5.6.40-nts-Win32-VC11-x64.zip】。

# 2. 解压 Php SDK

下载好的 SDK 是个压缩包，需要先解压出来。

# 3. 配置 SDK 环境变量

右键【此电脑】，选择【属性】。

在右侧找到并选择【高级系统设置】。

选择【高级】选项卡，然后点击【环境变量(N)...】

在下面【系统变量】中新增一个变量。

变量名输入【PHP_HOME】，然后变量值为 PHP SDK 的解压路径，然后点击确定。

找到【Path】变量，并【编辑】。

点击新建，值为【%PHP_HOME%】，然后点击【确定】。

点击【确定】。

# 4. 验证 SDK 是否配置正确

在命令行提示符中输入【php -version】。

出现以下提示，说明配置成功了。

如果出现以下提示，说明系统缺少 vc++11 环境，需要安装 vc++11。

下载地址：https://gitlab.com/toolcenter/tool/-/raw/main/vc-runtime-library/vc11/zh-cn/vcredist_x64.exe?inline=false (opens new window)

# 5. PhpStorm 中配置 PHP

点击【Settings】--》【Languages & FrameWorks】--》【PHP】。

确保 PHP language level 为【5.6】，选择【...】添加 PHP。

找到 PHP 的安装目录，然后选择【php.exe】。
