# SQL Server 允许外部连接

连接数据库时报错。

搞了半天，最后发现端口号写错了，默认端口号是 1433，要用 1433 作为默认端口号连接。

2023.02.18 测试发现，服务器上安装好 SQL Server 2019 之后就默认开启外网访问了，因此数据库密码一定要足够长，不要直接用其他服的，不然会被删库的！！

打开配置管理器。

找到【SQL Server 网络配置】，选择【MSSQLSERVER 的协议】，选择【TCP/IP】。

找到 IP 地址为【127.0.0.1】的一栏，然后把【已启用】改为【是】。

重新启动服务。
