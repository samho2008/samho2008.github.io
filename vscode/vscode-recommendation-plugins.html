# 排除 SC1017 报错

这个错误是说，尾行换行符应该用 \r，而不应该用 \r\n，或者 \n。但是使用 git 管理工具的话，是强制转换为 \r\n 的，所以要把这个设置关掉。

找到 ShellCheck 插件，点击小齿轮，找到【Extension Settings】

找到 Shellcheck: exclude，点击【Add Item】。

输入【1017】，然后点击【OK】

看到如图所示的效果说明修改成功了。
