# Windows 10 家庭版安装 IIS

左下角开始菜单输入【控制面板】，选择【程序】，点击【启用或关闭 Windows 功能】。

勾选【Internet Information Services】前面的勾，

依次展开【Internet Information Services】下的选项，除了 FTP 服务器不用勾选外，其他都选上。

注意：最后还要把【Internet Information Services 可承载的 Web 核心】勾选上。

最后勾选为如图所示的状态。

等待安装完成即可。

安装完成后点击【关闭】即可。

在浏览器中输入 http://localhost/ (opens new window) 并访问是否安装 IIS 成功。

看到以下界面说明配置成功了。

# Windows Server 2016 安装 IIS

安装环境：Windows Server 2016

  1. 打开【服务器管理器】，点击【添加角色和功能】。

  2. 进入【添加角色和功能向导】页面，点击下一步。

  3. 安装类型选择【基于角色或基于功能的安装】，点击下一步。

  4. 进入服务器选择页面，选择【从服务器池中选择服务器】，点击下一步。

  5. 进入服务器角色界面后，找到【Web 服务器(IIS)】并勾选上。

  6. 勾选后会自动弹出【添加角色和功能向导】界面，然后点击【添加功能】按钮即可，然后点击【下一步】。

  7. 选择安装的 .Net FrameWork 版本，按照下图的选择方式。即全选。然后点击下一步。

  8. 勾选完成后点击下一步。

  9. 选择角色服务，把【Web 服务器】全部选上，否则以后可能得重新安装。然后点击下一步。

  10. 点击【安装】，进行安装操作。

    如果安装的时候出现以下提示【WS-Managemnet 服务无法处理请求。该服务已配置为不接受任何远程 shell 请求。】

    则需要重启下服务器，重新进行上述步骤的安装。（所以建议如果使用的是阿里云服务器第一次进入的话，在安装 IIS 之前，先手动重启一遍）

  11. 安装成功后，点击【关闭】。

  12. 在浏览器中输入【127.0.0.1】, 出现下图，表示安装成功。

  13. 到控制面板→管理工具，找到【IIS】，使用即可。

# IIS 集成 PHP 7

下载【php-7.1.26-Win32-VC14-x64.zip】，下载地址：https://windows.php.net/downloads/releases/archives/php-7.1.26-Win32-VC14-x64.zip (opens new window)

注意：不要放到有中文、特殊字符的目录下解压，解压的时候注意选择【解压到 php-7.1.26-Win32-VC14-x64】

进入解压后的目录，找到【php.ini-production】复制一份新的，并重新命名为【php.ini】。

使用文本编辑器修改【php.ini】，并且搜索【extension_dir】。

找到如图所示的位置。

在下面新增一条记录，修改为当前 php 解压目录下的 ext 目录，注意要把所有的反斜杠改为向左的斜杠。

搜索【date.timezone】，在下面新增【date.timezone = Asia/Shanghai】。

搜索【fastcgi.impersonate】，在下面新增【fastcgi.impersonate = 1】。

搜索【cgi.rfc2616_headers】，在下面新增【cgi.rfc2616_headers = 1】。

搜索【cgi.force_redirect】，在下面新增【cgi.force_redirect = 0】，表示以 CGI 方式执行 PHP。

​ 打开 IIS，找到【处理程序映射】。

在右侧找到【添加模块映射...】。

在请求路径中输入【*.php】，模块选择【FastCgiModule】。

添加 PHP 可执行文件。

找到 PHP 的解压目录，右下角选择【*.exe】，找到【php-cgi.exe】。

最后在名称中输入【PHP】，点击确定。

选择【是】。

在左侧找到【网站】，右键【添加网站...】。

在 D 盘下新建目录【phpsite】，在里面新建一个文件【info.php】，输入以下内容：
