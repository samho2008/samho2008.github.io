# 系统级快捷键

快捷键 功能描述
Win + V 打开剪贴板管理器。注意是 Win + V，不是 Ctrl + V
Win + PS (PrintScreen) Windows 系统自带的截图工具，用于全屏截图。截图直接存放到 库中的 图片/屏幕截图 目录下。
Win + Shift + S Windows 系统自带的截图工具，可以全屏截图或区域截图，截图后放到画图中可以看到效果。
Win + I 打开 Windows 设置
Win + R (Runtime) 快速打开运行
Win + L (Lock) 快速锁屏
Win + E 打开「我的电脑」
Win + D 可以按「Ｗin + D」快速切换至桌面，再次按住「Ｗin + D」即可回到操作界面。

# 常用操作快捷键

快捷键 功能描述
Shift + 字母 临时输出字母的大写。效果等同于平时按住 Shift 键，但是来回切换大小写很麻烦，这个可以临时操作。
】 输入系统自带的输入法的时候，向后翻页。
【 使用系统自带的输入法的时候，向前翻页。
F11 全屏模式
Home (HM) 移动到当前光标所在的行首
END 移动到当前光标所在的行尾
Delete (DEL) 删除当前光标所在处后面的字符
Shift + ←/→ 从当前位置开始，向左/右选中
Alt + 双击 或 Alt + Enter 查看文件的属性。等同于先选中文件，然后右键属性(R)。
Ctrl + 双击 同时打开多个文件。比如想同时打开多个压缩包，可以先选中多个压缩包，然后用此快捷键。
Win + L 锁定桌面
Ctrl + W 关闭
Ctrl + A 全选
Ctrl + C 复制
Ctrl + V 粘贴
Ctrl + X 剪切。和复制的区别是，剪切之后原来的东西会消失，而粘贴不会。
Win + Q 打开搜索功能

Ctrl + W 关闭当前标签页

Ctrl + Shift + N 打开新的无痕模式窗口
Ctrl + T 打开新的标签页
Ctrl + U 打开当前网页的源代码
Ctrl + Tab 切换浏览器标签页
Ctrl + (+)/(-) 放大/缩小当前页面
F12 开发者工具
Ctrl + J 调出下载内容
Ctrl + H 调出浏览器历史记录
Ctrl + 鼠标左键 在新页面中打开链接
Ctrl + F 查找

# 虚拟桌面快捷键

快捷键 功能描述
Win + Tab 打开任务视图，切换虚拟桌面。
这个比Alt+Tab更容易用鼠标查找窗口。Alt+Tab松开手就没有了，Win键+Tab松开了，任务视图效果仍在，鼠标点击后才消失。
Win + Ctrl + D 创建新的虚拟桌面，并切换到它。
Win + Ctrl + ← 切换到相邻左侧的虚拟桌面。
Win + Ctrl + → 切换到相邻右侧的虚拟桌面。
Win + Ctrl + F4 删除当前虚拟桌面，切换到左侧桌面。
虚拟桌面删除后，其中的应用程序和文档会自动转到左侧的桌面中，不会丢失。

常见问题：

  • 应用会穿越虚拟桌面，点击之后会跳到其他的桌面中。

    在 WIN+TAB 的多任务界面中，选中微信，右键选择“在所有桌面中显示此应用窗口”，这样就不会来回跳了

参考文献：

90%的人不知道，如何高效使用Windows 10 虚拟桌面 - 知乎.html (opens new window)

# 切换窗口快捷键

快捷键 功能描述
Win + D 快速进入桌面，再次按【Win + D】恢复回来
Win + ⬅/➡ 分屏，可以使正在使用中的窗口贴向屏幕左侧/右侧，而只占用50% 的显示器面积，同时可继续用「Win+上下箭头」继续分屏。
ALT + TAB 切换窗口
Win + TAB 查看所有活动窗口

# 文件/文件夹快捷键

快捷键 功能描述
Ctrl + Shift + N 快速创建文件夹
Shift + DEL 快速删除文件/文件夹（不经过回收站，慎用！）
Ctrl + 移动鼠标 快速复制文件/文件夹，和【Ctrl + C】，【Ctrl + V】效果相同
Ctrl + Z 撤回上一步操作。比如误删除了一个文件，使用此快捷键可以快速恢复（强制删除不经过回收站则无法恢复！）
Ctrl + Y 恢复上一步操作。和 Ctrl + Z 配合使用。
Alt + ↑ 去到上级目录
Alt + ← 去到上一次所在的目录
F2 重命名。对着同一个文件慢慢点击两下也可以有此效果。
