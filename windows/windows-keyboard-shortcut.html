# Windows 系统的快捷键

# 系统级快捷键

快捷键 功能描述
Win + V 打开剪贴板管理器。注意是 Win + V，不是 Ctrl + V
Win + PS (PrintScreen) Windows 系统自带的截图工具，用于全屏截图。截图直接存放到 库中的 图片/屏幕截图 目录下。
Win + Shift + S Windows 系统自带的截图工具，可以全屏截图或区域截图，截图后放到画图中可以看到效果。
Win + I 打开 Windows 设置
Win + R (Runtime) 快速打开运行
Win + L (Lock) 快速锁屏
Win + E 打开「我的电脑」
Win + D 可以按「Ｗin + D」快速切换至桌面，再次按住「Ｗin + D」即可回到操作界面。

参考资料：

windows10下有哪些必须掌握的快捷键？ (opens new window)

# 常用操作快捷键

快捷键 功能描述
Shift + 字母 临时输出字母的大写。效果等同于平时按住 Shift 键，但是来回切换大小写很麻烦，这个可以临时操作。
F11 全屏模式
Home (HM) 移动到当前光标所在的行首
END 移动到当前光标所在的行尾
Delete (DEL) 删除当前光标所在处后面的字符
Shift + ←/→ 从当前位置开始，向左/右选中
Alt + 双击 或 Alt + Enter 查看文件的属性。等同于先选中文件，然后右键属性(R)。
Ctrl + 双击 同时打开多个文件。比如想同时打开多个压缩包，可以先选中多个压缩包，然后用此快捷键。
Win + L 锁定桌面
Ctrl + W 关闭
Ctrl + A 全选
Ctrl + C 复制
Ctrl + V 粘贴
Ctrl + X 剪切。和复制的区别是，剪切之后原来的东西会消失，而粘贴不会。
Win + Q 打开搜索功能

参考资料：

告别第三方APP：Win10的N种自带截图工具 (opens new window)

# 浏览器快捷键（主要针对 Chrome）

快捷键 功能描述
Ctrl + Shift + T 恢复已关闭的页面
F5 or Ctrl + R 刷新当前页面（但不会重新加载缓存）
Shift + F5 or Ctrl + F5 or Ctrl + Shift + R 刷新当前页面，并且忽略之前已加载的缓存
Ctrl + Shift + O 书签（网页收藏夹）
Ctrl + W 关闭当前标签页
Ctrl + N 打开一个新的浏览器窗口
Ctrl + Shift + N 打开新的无痕模式窗口
Ctrl + T 打开新的标签页
Ctrl + U 打开当前网页的源代码
Ctrl + Tab 切换浏览器标签页
Ctrl + (+)/(-) 放大/缩小当前页面
F12 开发者工具
Ctrl + J 调出下载内容
Ctrl + H 调出浏览器历史记录
Ctrl + 鼠标左键 在新页面中打开链接
Ctrl + F 查找

# 窗口操作快捷键

快捷键 功能描述
Alt + F4 关闭当前打开的（文件夹/浏览器等）窗口
Win + E 打开「我的电脑」窗口
Ctrl + N 复制出一个和当前文件夹一样的窗口

# 切换窗口快捷键

快捷键 功能描述
Win + D 快速进入桌面，再次按【Win + D】恢复回来
Win + ⬅/➡ 分屏，可以使正在使用中的窗口贴向屏幕左侧/右侧，而只占用50% 的显示器面积，同时可继续用「Win+上下箭头」继续分屏。
ALT + TAB 切换窗口
Win + TAB 查看所有活动窗口

# 文件/文件夹快捷键

快捷键 功能描述
Ctrl + Shift + N 快速创建文件夹
Shift + DEL 快速删除文件/文件夹（不经过回收站，慎用！）
Ctrl + 移动鼠标 快速复制文件/文件夹，和【Ctrl + C】，【Ctrl + V】效果相同
Ctrl + Z 撤回上一步操作。比如误删除了一个文件，使用此快捷键可以快速恢复（强制删除不经过回收站则无法恢复！）
Ctrl + Y 恢复上一步操作。和 Ctrl + Z 配合使用。
Alt + ↑ 去到上级目录
Alt + ← 去到上一次所在的目录
F2 重命名。对着同一个文件慢慢点击两下也可以有此效果。
